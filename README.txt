スマホ在庫管理 Pro 試作品

使い方
1. index.html をスマホまたはPCのブラウザで開く
2. 商品マスタで商品を登録
3. 読取画面でバーコードを読む、または番号を手入力
4. 棚卸確定・入庫・出庫・調整を選んで数量を保存
5. Excel出力ボタンで zaiko_stock.csv と zaiko_history.csv を出力

注意
- データはブラウザのlocalStorageに保存されます。同じ端末・同じブラウザ内のみ有効です。
- カメラ読取は Chrome/Edge 等の BarcodeDetector 対応ブラウザで動きます。
- iPhone Safariなど非対応の場合は、バーコード番号の手入力で使えます。
- 本格運用では、ログイン、クラウドDB、複数人同時編集、権限管理、バックアップを追加してください。

次に追加できる機能
- ログイン/担当者別権限
- 商品マスタのExcel取込
- 低在庫アラート
- 棚卸差異レポート
- 複数店舗・複数倉庫対応
- Googleスプレッドシート連携
